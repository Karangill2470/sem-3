RPS Game

Introduction:

In this project, I designed a simple console-based Rock, Paper, Scissors game using C#.
The game allows users to play against a computer opponent in the classic Rock, Paper, Scissors
challenge. The program employs fundamental concepts of C# programming, including user input,
conditional statements, loops, and random number generation.

Features:

1.User Interaction:

The program begins by prompting the user to choose between "ROCK," "PAPER," or "SCISSORS." User input is case-insensitive and validated to ensure a valid choice is made.

2.Computer Choice:

The computer's choice is randomly generated using the Random class in C#. It can select "ROCK," "PAPER," or "SCISSORS."

3.Determining the Winner:

The game logic determines the winner based on the classic Rock, Paper, Scissors rules. If the user and computer choices are the same, the game results in a tie. Win-lose scenarios are determined using conditional statements.

4.Play Again:

After each round, the user is given the option to play again. The game continues until the user decides not to play anymore..
